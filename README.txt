== Description ==

Metomic is the easiest and most beautiful way to comply with global privacy regulations.

Metomic helps make websites:

* GDPR and CCPA compliant
* Stand out with a beautiful and modern cookie notice
* Trustworthy by enabling visitors with transparency and control of their data

All in the easiest way possible. Here’s how it works:

1. Create a Metomic account, and we’ll automatically scan your website and detect which third parties are in use, include Google Analytics, Facebook, LinkedIn, etc.
2. Choose the colour, position, language (20 choices!) and features of your cookie notice.
3. Follow our documentation to block third party scripts before they run on your webpage.
4. Get real time stats on number of consents and declines per Micropolicy

To be GDPR compliant, it is important that you install Metomic correctly. Metomic will scan your website and automatically detect third party scripts, but you need to block these third parties, using our documentation.

== Installation ==

1. Upload the `metomic` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Metomic Settings page and link your plugin with Metomic

== Frequently Asked Questions ==

= What is a micropolicy? =

A micropolicy is a description of what data you use and how you use it for a specific use-case.

= How do I ensure correct implementation of Metomic? =

Follow the steps taken within the documentation above.

= How do I change the appearence of the cookie widget? =

Navigate to the appearence section of the Metomic Dashboard, found at https://app.metomic.io.
